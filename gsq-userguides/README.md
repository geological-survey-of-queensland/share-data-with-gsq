# How to share help customers share data with GSQ

Customers can share large files, or a lot of files, with GSQ through the following methods. Click the link to read the userguide for GSQ staff.

* [S3 to S3 data transfer](https://github.com/geological-survey-of-queensland/share-data-with-gsq/blob/master/gsq-userguides/s3-to-s3-gsq-userguide)
* [Azure Storage to S3 data transfer](https://github.com/geological-survey-of-queensland/share-data-with-gsq/blob/master/gsq-userguides/azure-storage-to-s3-gsq-userguide)
* [AWS Snowball data transfer](https://github.com/geological-survey-of-queensland/share-data-with-gsq/blob/master/gsq-userguides/snowball-gsq-userguide)
